# Proyecto web


### Instalar dependencias
```shell
    npm install
```
 

### Inicializar modo desarrollo
```shell
    npm run dev
```

### Contruir proyecto modo produccion
```shell
    npm run build
```

### Contruir proyecto modo preview
```shell
    npm run preview
```
