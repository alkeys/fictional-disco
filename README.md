# Agradesco a mi familia por el apoyo en este proyecto y mis amigos xd


### Instalar dependencias
```shell
    npm install
```
 

### Inicializar modo desarrollo
```shell
    npm run dev
```

### Contruir proyecto modo produccion
```shell
    npm run build
```

### Contruir proyecto modo preview
```shell
    npm run preview
```
